TOUR Application

Booking application for a tour to the castles

TOUR Application

TOUR Application is the final product of the MSc Computer Science project. This subject requires students to create an application from the ground up that allows general users to reserve a trip to the castle in the north of the United Kingdom.

Gallery

Description


This application's user interface and prototype were designed in Adobe Xd before being developed in Android Studio. The prototype illustrates how users can interact with the application.

The application was built using Android Studio. The application was implemented in two main parts. The first is the Front-end, which uses the XML language, and the second is the Back-end, which uses the JAVA language. It also included Google API to integrate the map into the application.
